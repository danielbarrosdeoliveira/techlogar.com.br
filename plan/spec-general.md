# General Specs

# 1. Overview

- Project Name: Techlogar

- Objective: A high-authority personal portal bridging Technology (ADS/Cloud) and Theology, focused on extreme performance, SEO, and minimalist design.

- Target Audience: Anyone interested in theology (biblical, systematic, prepared sermons) and those interested in using technology to enhance their theological studies, such as mobile apps, AI, etc.

# 2. Stack

- Technical Stack (Backend Agnostic)

- Framework: Nuxt.js 3+ (Vue 3 - Composition API).

- Styling: Tailwind CSS v4 (Core engine via @theme in CSS).

- Animation: Vue-Bits

- Colors: OKLCH system for perceptual fidelity, using the design-system.md file as the source of truth.

- Content: @nuxt/content (Markdown .md files as the data source).

- Deployment: SSG (Static Site Generation), to be hosted on a cPanel environment.

- Beauty: Prettier, EsLint, eslint-plugin-simple-import-sort, prettier-plugin-tailwindcss

# 3. Design System

- Strictly follow all specifications defined in the design-system.md file.

# 4. Planning

- The project must feature two distinct presentations:
  -- Landing Page
  -- Blog
